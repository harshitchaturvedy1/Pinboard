# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| Latest (`main`) | ✅ |
| Older releases | ❌ |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in Pinboard, please email us at **stemsproutsamerica@gmail.com**.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We'll respond within 72 hours and work with you on a fix and responsible disclosure timeline.

Since Pinboard is a nonprofit educational project, we greatly appreciate responsible disclosure from the security community.
