## [1.0.0] — Initial Release

### Added
- Block-based editor powered by Google Blockly
- 7 block categories: Structure, Pins, Control, Logic, Math, Serial, Variables
- Arduino C code generation
- MicroPython code generation (Raspberry Pi)
- Platform switcher (Arduino ↔ Raspberry Pi)
- Hardware module library: DHT11/22, Photoresistor, Button, Servo
- Hardware Setup panel with live sensor value display
- In-browser emulator for testing without hardware
- Live session mode with join codes
- Breakout challenge system for classroom groups
- Arduino C Preview panel (live code output)
- Serial Monitor
- WebSerial-based flashing
- Enter Code mode (switch to text editing)
